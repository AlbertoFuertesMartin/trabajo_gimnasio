# Gestión de Gimnasio - CRUD PHP
Proyecto para gestionar la asignación de alumnos a clases dirigidas.

## Archivos
* **index.php**: Listado de asistencias con nombres reales (INNER JOIN).
* **agregar.php**: Formulario para crear nuevas asistencias.
* **eliminar.php**: Script para borrar registros.
* **gimnasio.sql**: Base de datos completa para importar en phpMyAdmin.

## Configuración
1. Importar `gimnasio.sql` en MySQL.
2. Colocar los archivos en la carpeta del servidor local.
3. Configurar la conexión en los archivos PHP si el usuario/password es distinto a root.
